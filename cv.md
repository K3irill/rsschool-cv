## Code Examples

### 'x' marks the spot.

```javascript
function xMarksTheSpot(matrix) {
	let coords = []

	for (let x = 0; x < matrix.length; x++) {
		for (let y = 0; y < matrix[x].length; y++) {
			if (matrix[x][y] === 'x') {
				coords.push(x, y)
			}
		}
	}

	if (coords.length !== 2) {
		return []
	}

	return coords
}
```
